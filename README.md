
# Password Strength Checker

A Python-based password strength checker that analyzes how secure a password is.

## Features
- Checks password length
- Detects uppercase, lowercase, numbers, and special characters
- Compares passwords against common password lists (rockyou.txt and common.txt)
- Gives a score and feedback
- Allows users to retry multiple passwords

## Requirements
- Python installed
- rockyou.txt seprate installation, place in same folder as the script
- common.txt in the same folder as the script

## How to Run
1. Download or clone this repository
2. Place all files in the same folder
3. Open Command Prompt in that folder
4. Run: python PasswordChecker.py
